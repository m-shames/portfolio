# Michelle Shames — Research Portfolio
Researcher and data scientist specializing in the quantitative evaluation of 
criminal legal reform. This repository contains background information, code, 
and visualizations from select past and ongoing research projects. 
Supplementary CV documents are shared below.   
 
----

## Select Projects

### [Police Oversight & Accountability](https://github.com/m-shames/portfolio/tree/main/police-oversight): 
Dissertation research evaluating Chicago Civilian Review Board reforms using 
an interrupted time series design. Includes pipeline for a 26,000+ record database built 
from FOIA and public data, cleaning and validation scripts, causal modeling, and visualizations.

### [Solitary Confinement](https://github.com/m-shames/portfolio/tree/main/solitary-confinement): 
Summary of solitary confinement-related research including analyses supporting civil rights litigation 
and links to publications.
